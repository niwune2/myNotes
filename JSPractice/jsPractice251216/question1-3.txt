課題 1-3：偶数の合計と個数を同時に返す
何を作るか

数値の配列を受け取り、
偶数だけを対象にして「合計」と「個数」を同時に返す関数 を作りなさい。

要件

関数名は analyzeEvenNumbers とする

引数は数値の配列 1 つ

for 文（または for...of）を使って処理する

偶数判定は % を使う

戻り値は以下の形式の オブジェクト とする
{ sum: 数値, count: 数値 }

配列が空の場合、sum と count はどちらも 0 とする

元の配列は変更しないこと

動作イメージ

[1, 2, 3, 4, 5, 6] → { sum: 12, count: 3 }

[1, 3, 5] → { sum: 0, count: 0 }
