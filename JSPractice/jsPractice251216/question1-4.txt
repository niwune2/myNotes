課題 1-4：偶数と奇数を分けて返す
何を作るか

数値の配列を受け取り、
偶数と奇数をそれぞれ別の配列に分けて返す関数 を作りなさい。

要件

関数名は separateEvenOdd とする

引数は数値の配列 1 つ

for 文（または for...of）を使う

偶数・奇数の判定は % を使う

戻り値は以下の形式の オブジェクト とする
{ evens: 配列, odds: 配列 }

元の配列は変更しないこと

空配列が渡された場合も正しく動作すること

動作イメージ

[1, 2, 3, 4, 5, 6]
→ { evens: [2, 4, 6], odds: [1, 3, 5] }
