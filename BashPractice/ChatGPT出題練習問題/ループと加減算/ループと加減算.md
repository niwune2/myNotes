## ミニドリル（合計編）

### 1. 1から10までの合計

```bash
sum=0
for i in {1..10}; do
    sum=$((sum+i))
done
echo $sum
```

- 出力は **55** になるはず。

---

### 2. 偶数だけの合計

```bash
sum=0
for i in {1..10}; do
    if [ $((i%2)) -eq 0 ]; then
        sum=$((sum+i))
    fi
done
echo $sum
```

- 出力は **30**（2+4+6+8+10）。

## Bash のノットイコール（≠）

条件式の中で使います。

### 数値比較

```bash
if [ "$a" -ne "$b" ]; then
    echo "aはbと等しくない"
fi
```

- `-ne` = not equal（数値用）

### 文字列比較

```bash
if [ "$str1" != "$str2" ]; then
    echo "違う文字列です"
fi
```

- `!=` = not equal（文字列用）
