# Step 1：ダミーファイル作成スクリプト（課題 1）

## ✅ 課題：作業用ディレクトリを作り、連番＋日付つきのダミーファイルを生成せよ

次の仕様を満たす **Bashスクリプト** を `step1_make_dummy.sh` という名前で作ってください。

### 仕様

1. スクリプトを実行した場所に、作業用ディレクトリ `work_dummy` を作成する

   * すでに存在する場合は **エラーで止めず**、中にファイルを作れる状態にする

2. `work_dummy` の中に、以下2種類のファイルを **それぞれ 10個ずつ** 作成する

   * **空ファイル**：`report_YYYYMMDD_001.txt` 〜 `report_YYYYMMDD_010.txt`
   * **中身入りファイル**：`note_YYYYMMDD_001.log` 〜 `note_YYYYMMDD_010.log`

3. `YYYYMMDD` は **スクリプト実行日**（例：`20260127`）を使う

4. `.log` ファイルの中身は各ファイルで違う内容にする（最低1行）

   * 例：`note file 003 created on 20260127` のように **番号と日付が入る** こと

5. 連番生成には **for** と **seq** を使う（どちらでもOKだが、どちらかは必ず使う）

6. ファイル作成は **touch / リダイレクト / echo** のいずれか（複数使ってOK）

### 実行例（イメージ）

* `bash step1_make_dummy.sh` を実行すると `work_dummy/` ができて、`.txt` が10個、`.log` が10個並ぶ状態になる

---

## ヒント（答えは書きません）

* 日付：`date +%Y%m%d`
* 3桁ゼロ埋め：`printf "%03d" "$i"`
* 中身入りファイル：`echo "..." > ファイル名`
